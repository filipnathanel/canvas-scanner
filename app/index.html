A quick tutorial:

  • Select an image with [ Choose a file ] button
  • Add automation points ( right click on automation controller )
  • Start Scan
  • After the scan is finished you can save the generated File

Notes:

  • Supported Browsers:
    • Chrome, Firefox, Edge
  • Files:
    • Supported files: png/jpg/svg
  • Automation:
    • Right click on Automation Point to remove it
    • The point between tho Automation points is a slope controller
